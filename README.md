# WISEMEN-NEST-CORE

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# watch mode
$ pnpm run start:dev
```

## Test

```bash
# unit tests
$ pnpm run test:run
```
